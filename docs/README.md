### Running the app
* `npm start`
* GET: `/posts`
  * Returns a JSON object every post in it.
* POST: `/posts`
  * Creates a post and saves it to the database.
* PUT: `/posts/:id`
  * Updates and existing post in the database.
* DELETE: `/posts/:id`
  * Deletes an existing post in the database.
